Welcome to re[wave]link
=======================

.. add images here

**Relink** is a robust and powerful, Lavalink wrapper built for discord.py. 

Features
--------
* Full asynchronous design.
* Support for Lavalink v4+.
* Easy to use with discord.py commands and interactions.
* Optimized networking via ``curl_ccfi`` (optional) and ``msgspec``.
