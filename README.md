
ReLink is a robust and powerful Lavalink wrapper for [discord.py](https://github.com/Rapptz/discord.py).

This is a maintained fork of [WaveLink](https://github.com/PythonistaGuild/Wavelink).

### Features

- Full asynchronous design.
- Lavalink v4+ Supported with REST API.
- discord.py v2+ Support.
- Object orientated design with stateful objects and payloads.
- Fully annotated and complies with Pyright strict typing.

### Installation

**ReLink** requires **Python 3.12 or higher**.

Currently, the only way to install this package is via ``git``
